447. Number of Boomerangs

2017-05-05
LeetCode

题目描述：

Given n points in the plane that are all pairwise distinct, a “boomerang” is a tuple of points (i, j, k) such that the distance between i and j equals the distance between i and k (the order of the tuple matters).

Find the number of boomerangs. You may assume that n will be at most 500 and coordinates of points are all in the range -10000, 10000.

Example: Input: [[0,0],[1,0],[2,0]]

Output: 2

Explanation: The two boomerangs are [[1,0],[0,0],[2,0]] and [[1,0],[2,0],[0,0]]

代码实现：

class Solution {
public:
    int numberOfBoomerangs(vector<pair<int, int>>& points) {
        int ret=0;
        for(int i=0;i<points.size();++i){
            unordered_map<int,int> hm;
            for(int j=0;j<points.size();++j){
                int a=(points[j].first)-(points[i].first);
                int b=(points[j].second)-(points[i].second);
                hm[a*a+b*b]++;
            }
            for(auto iter=hm.begin();iter!=hm.end();++iter){
                ret+=iter->second*(iter->second-1);
            }
        }
        return ret;
    }
};
